JSON

Características:

  1. JSON = Javascript Object Notation;
  2. É um formato de representação de dados;
  3. Mais simples que XML, que é utilizado para fins parecidos;
  4. Utiliza o formato de chave e valor;
  5. É leve para ser enviado por requisições;
  6. Muito utilizado para API e também arquivos de configuração.

O JSON aceita diversos tipos de dados:

  • Strings - Ex: "Olá Mundo"
  • Numeros - Ex: 1 ou 12.54
  • Booleanos - true ou false
  • Arrays - Ex: [1,2,3]
  • Objetos - Ex: {"nome": "Matheus"}
  • Dados nulos - null

As strings, obrigatoriamente devem estar envolvidas por aspas duplas, e não aspas simples. Caso contrário não será um JSON válido.

JSON - Exemplo de um array de objetos

    
      [
        {
          "nome": "Matheus",
          "idade": 30,
          "esta_trabalhando": true,
          "detalhes_profissao": {
            "profissao": "programador",
            "empresa": "empresa X"
          },
          "hobbies": [
            "Programar",
            "Correr",
            "Ler"
          ]
        },
        {
          "nome": "João",
          "idade": 25,
          "esta_trabalhando": false,
          "detalhes_profissao": {
            "profissao": null,
            "empresa": null
          },
          "hobbies": [
            "Academia",
            "Jogar"
          ]
        }
      ]

Não é permitido colocar uma vírgula na última linha de um array ou de um objeto.

    
      const objs = [
        {
          nome: "Matheus",
          idade: 30,
          esta_trabalhando: true,
          detalhes_profissao: {
            "profissao": "programador",
            "empresa": "empresa X"
          },
          hobbies: [
            "Programar",
            "Correr",
            "Ler"
          ]
        },
        {
          "nome": "João",
          "idade": 25,
          "esta_trabalhando": false,
          "detalhes_profissao": {
            "profissao": null,
            "empresa": null
          },
          "hobbies": [
            "Academia",
            "Jogar",
          ]
        },
      ]

Assim não pode ser transferido. Tem que ser convertido em um texto em JSON

Converter objeto para texto em JSON

    
      const jsonData = JSON.stringify(objs);

Saída: 

    
      [{"nome":"Matheus","idade":30,"esta_trabalhando":true,"detalhes_profissao":{"profissao":"programador","empresa":"empresa X"},"hobbies":["Programar","Correr","Ler"]},{"nome":"João","idade":25,"esta_trabalhando":false,"detalhes_profissao":{"profissao":null,"empresa":null},"hobbies":["Academia","Jogar"]}]

Converter texto no formato JSON para objeto

    
      const objData = JSON.parse(jsonData);

Trabalhando com os dados como um objeto

    
      objData.map((pessoa) => {
        console.log(pessoa.nome);
      });
